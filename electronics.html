What is an ISP?

ISP stands for In-System Programmer. It allows firmware updates to be delivered to the on-chip memory of microcontrollers and related processors without requiring specialist programming circuitry on the circuit board, and simplifies design work.

What is a firmware?

As the name suggests, FIRMware is a special software that is permanently burned or embedded onto hardware device by the vendor. In other words, it is a software designed for a specific hardware. Once the firmware is burned aka uploaded into the chip, it will be permanently stored within the chip in the form of Read-Only-Memory (ROM). A firmware acts as the intermediary between the hardware and the software. It provides intructions how the hardware should function. So what is the difference between software and firmware? Firmware just handles very low level functions without which the device will become non-functional however software is used to address high level applications.

How to prepare your files for 2D Milling?

Step 1: Save a picture of your PCB diagram as a PNG file.
Step 2: Open up Mods.
Step 3: From Mods, under settings, click Programs >> Open Server Programs >> Mill 2D PCB PNG.
Step 4: Click here to get traces settings.
Step 5: Click here to get outline settings.
Step 6: After configuring the settings, click Calculate. This will convert the PNG file to NC file which is the G-code.
Step 7: To simulate the milling operation, open up CAMotics software. Import your NC file into the software.
Step 8: Under the 'Simulation' tab, click Run, then click Play. This will simulate the milling operation. Below is an example of the ISP traces Simulation in the CAMotics software.


How to use EAGLES software to design your PCB?

Step 1: Download EAGLES software.
Step 2: Download fab_lbr.
Step 3: Copy the fab_lbr into your library folder. This folder directory is Documents >> EAGLE >> Libraries.
Step 4: Open up your EAGLES software. On the left hand side, click on the Projects Tab, then right click on the project folder. Then, create new project. Once new project folder has been created, right click on the new project folder and create 'schematics'.
Ensure that your fab_lbr has been downloaded into your EAGLE library. Now that your workspace is ready, we will try to design the circuit that is shown below. The example circuit drawn below is an ATtiny45 Hello World Board. Click here to obtain the schemetics.

Step 5: Use the 'Add Part' tool on the left hand side to add all the parts into the schematics. Use ' * ' to accsss the parts. For example, type *FRAME* in the search bar to obtain the drawing frame of the circuit design.
Step 6: Use the 'Net' tool on the left side to create all of the connections in the circuit diagram.
Step 7: To move the parts, bring your cursor to the plus sign in the object and move around. And to rotate the part, repeat the same steps as before while doing a right click. Below are all the list of parts found in the eagle library that needs to be added into the cicuit for ATtiny45 Hello World Board.

  • UC_ATTINY45SI (Atmel Chip)
  • CONN_03X2_AVRISPSMD (AVRISP SMD)
  • CONN_06_FTDI-SMD-HEADER (FTDI chip)
  • FRAME
  • CAP_UNPOLARIZEDFAB (Capacitor)
  • R1206FAB (Resistor)
  • LEDFAB1206 (LED)
  • Vcc
  • Gnd
